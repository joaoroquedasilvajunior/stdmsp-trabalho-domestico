Incorporar gráficos no seu site

Cada gráfico abaixo pode ser incorporado em qualquer site através de um iframe. Os dados são atualizados automaticamente — você cola o código uma vez e ele acompanha as mudanças no painel.

Como funciona

  1. Escolha o gráfico que você quer incorporar.
  2. Clique em "Copiar código" — o iframe é copiado para a área de transferência.
  3. Cole o código no editor HTML do seu site (Wix, WordPress, página HTML, etc.). A maioria dos editores tem um bloco "HTML personalizado".
  4. O gráfico se atualiza sozinho: cada vez que o painel ganha dados novos, seu iframe mostra as novidades automaticamente.

Os iframes incluem uma pequena atribuição "fonte: Painel Trabalho Doméstico" no canto inferior. Atribuição é importante para credibilidade — pedimos que você não a remova.
